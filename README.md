# CORD Creator Calender: STEAM GAMES

Static GitHub Pages package for the public Steam games dashboard.

## Files the worker machine updates

- `data/steam/worker-feed/latest.json`
- `data/steam/horror-upcoming/worker-feed/latest.json`
- Optional manual curation file: `data/steam/gamepass-games.json`

The site is static. It does not need the local backend and should not publish the full private repo.
